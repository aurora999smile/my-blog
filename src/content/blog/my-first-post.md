# 如何搭建个人博客

## 为什么选择 Astro？

- ⚡ 速度快
- 🎨 简单易用
- 📝 原生支持 Markdown

## 开始之前你需要

1. Node.js
2. 代码编辑器
3. 基本的命令行知识

## 第一步：安装 Astro

```bash
npm create astro@latest my-blog
```

就这么简单！开始你的博客之旅吧 🚀
